Getting Started

Install the SDK and initialise a client in seconds

1  — Install
pip install authentasdk
2  — Create the client
from authenta import AuthentaClient

client = AuthentaClient(
    base_url="https://platform.authenta.ai",
    client_id="YOUR_CLIENT_ID",
    client_secret="YOUR_CLIENT_SECRET",
)

client_secret

Liveness · Face Swap · Similarity

Service 1 — Liveness Detection

Is this face real and live, or a presentation attack?

  • Printed photos held in front of a camera
  • Screen replays of face videos
  • Deep fake video injections

Image Video   →  isLiveness: true / false

Liveness — Real Video

API call
media = client.face_intelligence(
    path="face_live_video/real/3.mp4",
    model_type="FI-1",
    livenessCheck=True,
)
✅   Live Face: YES

Liveness — Presentation Attack

API call
media = client.face_intelligence(
    path="face_live_video/fake/4.mp4",
    model_type="FI-1",
    livenessCheck=True,
)
● Live Face: NO

Liveness — Image Input

API call
media = client.face_intelligence(
    path="face_live_images/real/1.jpg",
    model_type="FI-1",
    livenessCheck=True,
)
✅   Live Face: YES

Service 2 — Face Swap Detection

Was this face AI-generated or swapped onto the video?

  • DeepFaceLab, FaceSwap, Roop, and similar tools
  • Neural face reenactment
  • Synthetic face injection

Video only   →  isDeepFake: true / false

Face Swap — Manipulated Video

API call
media = client.face_intelligence(
    path="faceswap/fake/1.mp4",
    model_type="FI-1",
    faceswapCheck=True,
)

API call

Service 3 — Face Similarity

Do these two images belong to the same person?
